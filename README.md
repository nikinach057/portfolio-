# Nachappa PP - Portfolio Website

A modern, responsive portfolio website showcasing professional experience in Smart Manufacturing, AI/ML, and Software Development.

## Project Structure

```
portfolio/
├── index.html                 # Main HTML file
├── css/                      # Stylesheets
│   ├── base.css             # Base styles and typography
│   ├── animations.css       # Animation keyframes and effects
│   ├── components.css       # Component-specific styles
│   └── bullets.css          # Custom bullet designs
├── js/                      # JavaScript files
│   ├── mobile-menu.js       # Mobile menu functionality
│   ├── smooth-scroll.js     # Smooth scrolling behavior
│   ├── animations.js        # Intersection Observer animations
│   └── navigation.js        # Navigation highlighting
├── components/              # HTML section components
│   ├── header.html          # Navigation header
│   ├── hero.html            # Hero section
│   ├── projects.html        # Featured projects
│   ├── skills.html          # Technical skills
│   ├── about.html           # About section
│   ├── experience.html      # Professional experience
│   ├── contact.html         # Contact information
│   └── footer.html          # Footer
├── assets/                  # Static assets
│   └── images/              # Company logos and images
│       ├── ADL_Logo.png
│       ├── BHEL.png
│       ├── cmti.png
│       └── cognizant.png
└── README.md               # This file
```

## Features

- **Modular Architecture**: Clean separation of HTML, CSS, and JavaScript
- **Responsive Design**: Mobile-first approach with Tailwind CSS
- **Smooth Animations**: Intersection Observer-based fade-in effects
- **Interactive Navigation**: Smooth scrolling and active section highlighting
- **Mobile Menu**: Collapsible navigation for mobile devices
- **Component-Based**: Reusable HTML components for easy maintenance

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Custom styles with animations and transitions
- **JavaScript (ES6+)**: Modern JavaScript for interactivity
- **Tailwind CSS**: Utility-first CSS framework
- **Font Awesome**: Icon library
- **Google Fonts**: Montserrat typography

## Customization

### Adding New Sections
1. Create a new HTML file in the `components/` directory
2. Add the component to the `index.html` component loader
3. Style the component in the appropriate CSS file

### Modifying Styles
- Base styles: `css/base.css`
- Animations: `css/animations.css`
- Components: `css/components.css`
- Custom elements: `css/bullets.css`

### Adding JavaScript Functionality
- Create new JS files in the `js/` directory
- Import them in `index.html`
- Follow the existing modular pattern
